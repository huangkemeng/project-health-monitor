## 7.1 性能需求

### 7.1.1 响应时间

| 场景 | 目标值 | 测量方法 |
|-----|:------:|---------|
| 页面加载时间 | < 2秒 (95%请求) | 从请求到首屏渲染完成 |
| API响应时间 | < 500ms (95%请求) | 从请求到响应返回 |
| 登录响应时间 | < 1秒 | 从提交到跳转完成 |
| 监控大盘加载 | < 1.5秒 | 统计数据+列表数据 |
| 探测执行时间 | < 30秒 | 单次探测完整执行 |

### 7.1.2 吞吐量

| 指标 | 目标值 | 说明 |
|-----|:------:|------|
| 最大并发探测数 | 100 | 同时执行的探测任务 |
| API QPS | 100 | 每秒请求数 |
| 支持的监控项数 | 1000 | 单实例支持的最大监控项 |
| 日探测次数 | 100万 | 按1000监控项*60秒间隔计算 |

### 7.1.3 资源使用

| 资源 | 目标值 | 说明 |
|-----|:------:|------|
| CPU使用率 | < 70% | 正常运行时 |
| 内存使用 | < 2GB | 单实例 |
| 磁盘I/O | < 50MB/s | 峰值 |
| 网络带宽 | < 10Mbps | 出站流量 |

## 7.2 可用性需求

### 7.2.1 系统可用性

| 指标 | 目标值 | 计算方式 |
|-----|:------:|---------|
| 服务可用率 | > 99.5% | (总时间-故障时间)/总时间 |
| 计划维护时间 | < 4小时/月 | 预先通知的维护窗口 |
| 故障恢复时间 | < 5分钟 | 从故障发现到恢复 |
| 数据备份频率 | 每日 | 自动备份 |

### 7.2.2 故障处理

| 故障类型 | 处理策略 | RTO | RPO |
|---------|---------|:---:|:---:|
| 服务进程崩溃 | 自动重启 | 1分钟 | 0 |
| 服务器宕机 | 手动恢复 | 30分钟 | 5分钟 |
| 数据库损坏 | 从备份恢复 | 1小时 | 1天 |
| 网络中断 | 等待恢复 | - | 0 |

## 7.3 安全需求

### 7.3.1 认证与授权

| 需求 | 实现方式 |
|-----|---------|
| 密码加密 | bcrypt算法，cost factor 12 |
| Token安全 | JWT HS256签名，7天过期 |
| 传输加密 | 全站HTTPS，TLS 1.2+ |
| 会话管理 | 支持强制登出 |

### 7.3.2 数据安全

| 需求 | 实现方式 |
|-----|---------|
| 敏感数据加密 | 密码哈希存储 |
| 数据备份加密 | 备份文件加密存储 |
| 数据访问控制 | 行级数据隔离 |
| 审计日志 | 记录关键操作 |

### 7.3.3 防护机制

| 威胁 | 防护措施 |
|-----|---------|
| SQL注入 | 使用ORM参数化查询 |
| XSS攻击 | 输入过滤，输出转义 |
| CSRF攻击 | Token验证 |
| 暴力破解 | 登录失败锁定 |
| DDoS攻击 | 请求频率限制 |
