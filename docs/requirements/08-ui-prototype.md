## 8.4 交互规范

### 8.4.1 通用交互

| 操作 | 反馈 |
|-----|------|
| 点击按钮 | 按钮置灰，显示loading |
| 表单提交 | 验证通过后才提交 |
| 删除操作 | 弹出确认对话框 |
| 保存成功 | Toast提示，自动跳转 |
| 操作失败 | Toast提示错误信息 |

### 8.4.2 状态颜色

| 状态 | 颜色 | 色值 |
|-----|------|------|
| 正常 | 绿色 | #52c41a |
| 警告 | 黄色 | #faad14 |
| 严重 | 红色 | #f5222d |
| 暂停 | 灰色 | #bfbfbf |
| 信息 | 蓝色 | #1890ff |
